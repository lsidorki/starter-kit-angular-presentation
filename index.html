About Angular (v. 4.0.0)

  • Smaller & Faster (will be still improved)
  • View Engine changes (generates less code)
  • Animation Package (moved to own package)
  • Improved *ngIf (if/else syntax) and *ngFor (assign local variables)
  • Angular Universal (allows developers to run Angular on a server)
  • TypeScript 2.1 and 2.2 compatibility
  • Packaging Changes
